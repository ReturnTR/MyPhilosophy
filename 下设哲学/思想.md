# 什么是思想

思维的内容，真、善、美的具体形式。简单化来说就是人是怎么想的，包括：

- 善：什么是重要的，自己追求的是什么

- 真：这个世界是什么样的

- 美：什么是好看的，值得保存下来的

如何分析：从历史的角度看
